# Laravel development

## Kdy aktivovat

Úpravy v `app/`, `routes/`, `database/`, `config/`, `tests/` (PHP).

## Postup

1. `docs/PROJECT-PROFILE.md` — verze Laravel, test framework.
2. Sibling soubory — kopíruj styl (array vs string validation rules).
3. `php artisan make:* --no-interaction` pro nové třídy.
4. `./vendor/bin/pint` před dokončením.
5. `php artisan test --compact` pro dotčenou oblast.

## Laravel Boost

Pokud je nainstalován: `search-docs` před neznámým API ekosystému.
