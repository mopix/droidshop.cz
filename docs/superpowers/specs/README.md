# Specifikace (zadání)

Závazné zadání funkcí a oblastí. Změna spec = vědomá změna požadavků.

## Pojmenování

`YYYY-MM-DD-nazev.md` — např. `2026-05-17-export-csv.md`

## Šablona nového spec

```markdown
# <Název feature / oblasti>

**Datum:** YYYY-MM-DD  
**Status:** draft | approved | in-progress | done | superseded  
**Související plán:** `docs/superpowers/plans/YYYY-MM-DD-nazev.md` (pokud existuje)

## Kontext

Proč to stavíme, pro koho.

## Cíle

- [ ] Cíl 1
- [ ] Cíl 2

## Mimo rozsah

- Co explicitně nestavíme.

## Požadavky

### Backend
...

### Frontend
...

## Akceptační kritéria

1. ...
2. ...

## Technické poznámky

Odkazy na API, modely, route názvy.

## Reference

- As-is (po dokončení): `docs/as-is/YYYY-MM-DD-nazev.md`
```
